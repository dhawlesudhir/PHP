  1. What is OOP ?
    • coding methodology / style / pattern
    • code more reusable / modern
    • easy to maintain and understand
    • best for medium and large projects
      Frameworks?
      Laravel, Codeigniter, Cake PHP, Yii, Zend, Symphony
  2. Whats is Constructor Function ?
    • Autocalled function
    • Whenever an object of class gets created Constructor functioned get called
    • used to initial work such as initializating varibales or resources like DB connections
    • '__construct()' predefined funtion name
  3. Access Modifier ?
    • restictions on varibles and methods of an object
    • 3 types
    • public - accessable using object of the class
    • private - accessable only within class
    • protected - accessable to within class and inheriated classes
  4. Overriding Properties & Methods?
    • derived class has same named Properties or methods, then values of parent/base class will get replaced by derived class
      • Advantages of method overriding
    • Helps in writing generic code based on parent class or interface as object resolution happens at runtime
    • Provides multiple implementation of same method and can invoke parent class overridden method using super keyword
    • Defines what behavior a class can have and implementation of behavior has been taken care by class which is going to implement
  5. Namespace?
    • Avoiding name collisions between classes/functions/constants defined by someone with third-party classes/functions/constants.
    • Provides ability to alias (or shorten) Extra_Long_Names thereby improving readability of source code
